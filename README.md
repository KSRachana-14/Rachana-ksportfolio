# 🚀 Professional Client Portfolio

A modern, responsive, and high-performance personal portfolio website built with **HTML, CSS, and Vanilla JavaScript**. This project is designed to be lightweight, professional, and easy to deploy without needing any complex development environments.

## ✨ Features

- **🏠 Homepage (Hero)**: Modern landing section with a profile photo and clear Call-to-Action (CTA) buttons.
- **👤 About Section**:
  - **About Me**: Professional summary.
  - **Education**: Structured timeline of academic history.
  - **Skills**: Categorized technical skills and tools.
  - **Achievements**: Interactive cards that open detailed **modal popups**.
- **📜 Certifications**: Grid of certificates with a built-in **modal previewer**.
- **💼 Projects**: Featured work cards with tech stack badges and links to live demos/GitHub.
- **📄 Resume Functionality**: Dedicated buttons for both downloading a PDF and viewing it online.
- **📱 Responsive Design**: Fully mobile-friendly with a functional hamburger menu.
- **⚡ Smooth Interactions**: Scroll-smooth navigation and transition animations.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure.
- **Tailwind CSS (CDN)**: Utility-first styling for a modern UI.
- **Vanilla JavaScript**: Core functionality for modals, navigation, and scroll effects.
- **Lucide Icons**: Crisp, professional SVG icons.
- **Google Fonts**: "Inter" font for clean typography.

## 🚀 Getting Started (Local)

Since this project uses plain HTML/JS, you don't need to install anything!

1.  **Download** or clone this repository.
2.  **Open** the `index.html` file in any web browser (Chrome, Firefox, Edge, etc.).
3.  **To Edit**: Open `index.html` in any text editor (VS Code, Notepad++, etc.) to change the content.

## 🎨 Customization

To personalize the portfolio for your client:

1.  **Personal Info**: Search for "Alex Rivera" in `index.html` and replace it with the client's name.
2.  **Profile Photo**: Update the `<img>` src in the Hero section.
3.  **Resume**: Place the client's resume PDF in the root folder and name it `resume.pdf`.
4.  **Projects & Certs**: Update the text and image URLs in the respective sections.

## 🌐 Deployment (Netlify)

This project is optimized for **Netlify** deployment:

### Option 1: Netlify Drop (Easiest)
1.  Go to [Netlify Drop](https://app.netlify.com/drop).
2.  Drag and drop your project folder onto the page.
3.  Your site is live!

### Option 2: GitHub Integration
1.  Push your code to a GitHub repository.
2.  Connect the repository to Netlify.
3.  **Build Settings**:
    - **Build Command**: (Leave empty)
    - **Publish Directory**: `.` (or the root folder)
4.  Click **Deploy site**.
